# n8s

docker in Node.js


## How to start
```shell
yarn
yarn watch
```

```shell
yarn build
yarn start
```


## 镜像、容器

### 创建基础镜像
```shell
docker build -t n8s:1.0.0 .
```

### 创建并后台运行容器
```shell
docker run -p 9000:9000 -itd --name liuyi_dt-tag n8s:1.0.0
```

### 进入容器
```shell
docker exec -it liuyi_dt-tag /bin/sh
```
